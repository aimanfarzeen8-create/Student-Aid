MedTerms – AI-Powered Study Companion

Problem It Solves :
Medical students often struggle with organizing study schedules, maintaining focus, and reviewing complex terminology.
MedTerms provides an integrated solution: a planner, focus timer, medical dictionary, and quiz system —
all in one app — to help students learn efficiently and stay consistent.

 Live URL :
Click here to use the app(https://student-aid--aimanfarzeen8.replit.app)

Features :
Planner – Organize study tasks with categories, priorities, and progress tracking.
Focus Timer– Pomodoro, deep study, and calm breathing modes to improve concentration.
Dictionary – Search and filter medical terms by specialty and difficulty level.
Quiz Mode– Self-testing with flashcards and quizzes to reinforce learning.
Progress Tracker– Visual dashboard showing study completion and goals.

AI Feature :
- Integrated AI-driven quiz generator that adapts questions based on user progress.  
- Uses a custom system prompt to generate clear, beginner-friendly explanations of medical terms.  
- Example system prompt:  
  “Explain this medical term in simple words suitable for a first-year medical student, and generate a short quiz question with answer choices.”
